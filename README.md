## ğŸ“¦ Installation

### Prerequisites

- Bash 4.0 or later
- Git
- Basic system administration knowledge

### Installation Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/tiation/tiation-laptop-utilities.git
   cd tiation-laptop-utilities
   ```

2. **Make scripts executable**
   ```bash
   chmod +x setup.sh
   chmod +x scripts/*.sh
   ```

3. **Run the setup**
   ```bash
   ./setup.sh
   ```
